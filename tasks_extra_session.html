Extra Tasks: Session Related

Tasks

The following are simple session related tasks, useful for closing or locking a session:

  • Lock Session: locks the session and waits for credentials, but all applications remain open.

  • Log Off: closes all open applications and exits the current session, opening the login screen.

  • Hibernate: (Windows only) takes the workstation into hibernated state, without closing any application.

  • Shutdown: closes all open applications and powers off the workstation.

  • Reboot: closes all open applications and restarts the workstation.

In all cases the task definition form is similar to the one below.

These tasks do not require any extra parameter.
